adjustAudioMixingVolume method

  1. @override
Future<void> adjustAudioMixingVolume (
  1. int volume
)

Adjusts the volume of audio mixing. Call this method when you are in a channel. Note

  • Calling this method does not affect the volume of the audio effect file playback invoked by the playEffect method. See RtcEngine.playEffect Param volume Audio mixing volume. The value ranges between 0 and 100 (default).

Implementation

@override
Future<void> adjustAudioMixingVolume(int volume) {
  return _invokeMethod('adjustAudioMixingVolume', {'volume': volume});
}
